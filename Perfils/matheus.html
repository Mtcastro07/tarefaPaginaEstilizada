Catalogo de filmes

Plataforma copia do imdb

Descrição:

Este projeto foi desenvolvido como trabalho final da disciplina de Desenvolvimento Web do Instituto Federal do Rio de Janeiro (IFRJ) e consiste em um catálogo de filmes inspirado em plataformas como o IMDb. A aplicação foi construída com React no frontend, utilizando componentes reutilizáveis e uma interface dinâmica, enquanto o Supabase foi empregado para o gerenciamento e armazenamento dos dados. O projeto permitiu aplicar na prática conceitos de desenvolvimento web moderno, integração entre frontend e backend.

Metodologia:

Durante o desenvolvimento do projeto, foi adotada uma abordagem baseada em componentes utilizando React, permitindo a criação de uma interface organizada, reutilizável e de fácil manutenção. Para o armazenamento e gerenciamento dos dados da aplicação, foi utilizado o Supabase, enquanto a obtenção das informações dos filmes, como títulos, sinopses, avaliações e pôsteres, foi realizada por meio da API do TMDB (The Movie Database). A integração dessas tecnologias possibilitou a construção de uma aplicação dinâmica, responsiva e alinhada às práticas atuais do desenvolvimento web.
